CampusLens

Real-time campus object detection and multi-object tracking system deployed on edge hardware. CampusLens identifies people, vehicles, and facilities at two campus gates using an anchor-free YOLOv8 backbone with custom Bangladeshi campus fine-tuning.

Overview

CampusLens started as an effort to address real safety and logistics challenges on our university campus. The system leverages an anchor-free multi-object tracking (MOT) pipeline optimised for resource-constrained edge devices — Raspberry Pi 5 and NVIDIA Jetson Nano — without sacrificing accuracy.

The backbone is a custom-trained YOLOv8n model fine-tuned on a dataset of 12,000 images collected across three Bangladeshi university campuses, covering pedestrians, rickshaws, motorcycles, bicycles, and campus-specific objects such as guard booths and bulletin boards.

Architecture

  1. 01

    Frame Ingestion

    RTSP streams from IP cameras are decoded at the edge node. Frames are downscaled to 640×640 and batched for the detector.

  2. 02

    Detection — YOLOv8n

    Anchor-free detection head predicts bounding boxes and class probabilities across 8 classes. INT8 quantisation cuts model size from 6.3 MB to 1.8 MB and doubles throughput on Jetson.

  3. 03

    Tracking — ByteTrack

    ByteTrack associates detections across frames using both high- and low-confidence boxes, recovering occluded objects that naive trackers miss.

  4. 04

    Dashboard & Alerts

    A lightweight Flask dashboard streams annotated video to browser clients via WebSocket. Configurable crowd-density thresholds trigger real-time alerts.
