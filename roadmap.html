Your ML journey

Learning Roadmap

Structured, opinionated learning paths from zero to research-ready. Pick a track and follow the phases — or start with Foundations if you are new to ML.

The essential foundation every ML practitioner needs — regardless of specialisation. Complete this track before branching into any other.

🕐 Est. 14–20 weeks
01

Prerequisites

2 – 4 weeks

Build the mathematical and programming backbone you will rely on throughout your ML career.

Python for Data Science

NumPy, Pandas, Matplotlib — the core scientific stack

📚 Python Data Science Handbook 🎤 CS50P (free)

Linear Algebra

Vectors, matrices, eigenvalues, SVD — the geometry behind ML

🎬 3Blue1Brown — Essence of Linear Algebra 📚 Gilbert Strang — Intro to Linear Algebra

Statistics & Probability

Distributions, Bayes' theorem, hypothesis testing, MLE

🎬 StatQuest with Josh Starmer 📚 Think Stats (free online)

Calculus & Optimisation

Derivatives, chain rule, gradient descent intuition

🎬 3Blue1Brown — Essence of Calculus
02

Classical Machine Learning

4 – 6 weeks

Core ML algorithms, evaluation methodology, and hands-on practice with scikit-learn.

Supervised Learning

Linear & logistic regression, SVMs, decision trees, random forests

🎬 Andrew Ng — ML Specialisation (Coursera) 📚 Hands-on ML with Scikit-Learn

Unsupervised Learning

K-means, DBSCAN, PCA, t-SNE, autoencoders

🎬 Stanford CS229 Lecture Notes

Model Evaluation & Selection

Cross-validation, bias-variance tradeoff, precision/recall, ROC-AUC

📚 scikit-learn documentation

Feature Engineering

Encoding, scaling, imputation, feature selection, pipelines

🎬 Kaggle — Feature Engineering Course (free)
03

Deep Learning

6 – 8 weeks

Neural networks from scratch, modern frameworks, and the training loop in full detail.

Neural Networks & Backpropagation

Forward/backward pass, activation functions, weight initialisation

🎬 Andrej Karpathy — Neural Networks: Zero to Hero

PyTorch Fundamentals

Tensors, autograd, nn.Module, DataLoader, training loop

🎬 fast.ai Practical Deep Learning (free) 📚 Official PyTorch tutorials

Regularisation & Optimisation

Dropout, batch norm, Adam, learning rate schedules, early stopping

🎬 deeplearning.ai — Improving Deep Neural Networks
04

Choose Your Specialisation

Next track

You now have the foundation to dive deep into any specialisation.

From pixel operations to state-of-the-art detection and generation models. Assumes ML Foundations is complete.

🕐 Est. 16–22 weeks
01

Image Basics & CNNs

3 – 4 weeks

Image representations, classical operations, and the core convolutional building block.

Image Processing with OpenCV

Colour spaces, morphological ops, edge detection (Sobel, Canny), histograms

📚 OpenCV Python Tutorials (official)

Convolutional Neural Networks

Conv layers, pooling, receptive field, LeNet → AlexNet intuition

🎬 Stanford CS231n — CNNs for Visual Recognition

Transfer Learning

Fine-tuning pretrained models (ResNet, EfficientNet) on custom datasets

🎬 fast.ai Lesson 1–3 📚 Torchvision pretrained models
02

Object Detection & Segmentation

5 – 7 weeks

Locate and classify multiple objects; pixel-level understanding of images.

Anchor-Based Detection

Faster R-CNN, anchor boxes, IoU, non-max suppression, RPN

📄 Paper: "Faster R-CNN" (Ren et al. 2015)

YOLO Family

YOLOv5 → YOLOv8, anchor-free detection, custom training pipeline

🎬 Ultralytics YOLOv8 Docs

Semantic & Instance Segmentation

U-Net, Mask R-CNN, SAM (Segment Anything Model)

📄 Paper: "U-Net" (Ronneberger et al. 2015) 🎬 Meta AI SAM tutorial
03

Vision Transformers & Generative Models

5 – 7 weeks

Modern architectures that push the state of the art in recognition and generation.

Vision Transformers (ViT)

Patch embeddings, self-attention for images, DINO, DINOv2

📄 Paper: "An Image is Worth 16×16 Words" (Dosovitskiy et al.)

Diffusion Models

DDPM, score matching, latent diffusion, fine-tuning with LoRA / DreamBooth

🎬 HuggingFace Diffusers course (free) 📄 Paper: "DDPM" (Ho et al. 2020)

Multimodal Models

CLIP, BLIP-2, image-text contrastive learning, visual question answering

📄 Paper: "CLIP" (Radford et al. 2021)
04

Production & Research

Ongoing

Deploy models to real systems, contribute to open-source, or pursue original research.

Edge Deployment

ONNX export, TensorRT, OpenVINO, model quantisation & pruning

Read & Reproduce Papers

Pick papers from CVPR / ICCV / ECCV and implement them from scratch

Text processing, sequence modelling, and modern LLMs — from bag of words to fine-tuning billion-parameter models. Assumes ML Foundations is complete.

🕐 Est. 16–24 weeks
01

Text Processing & Classical NLP

3 – 4 weeks

The building blocks that underpin every modern NLP system.

Text Preprocessing

Tokenisation, stemming, lemmatisation, stopwords, regex, spaCy

📚 spaCy 101 (official docs)

Bag-of-Words & TF-IDF

CountVectorizer, TF-IDF, Naive Bayes classifier, text similarity

🎬 Natural Language Processing with Python (NLTK Book)

Word Embeddings

Word2Vec, GloVe, FastText — dense representations of meaning

🎬 Stanford CS224N — Word Vectors lecture
02

Sequence Models & Attention

4 – 6 weeks

Modelling order and context in language — the path to Transformers.

RNNs & LSTMs

Vanishing gradients, gated units, sequence-to-sequence, teacher forcing

🎬 Andrej Karpathy — char-RNN blog post

Attention Mechanism

Bahdanau attention, scaled dot-product attention, multi-head attention

📄 Paper: "Attention Is All You Need" (Vaswani et al. 2017) 🎬 The Illustrated Transformer — Jay Alammar
03

Transformers & Pretrained LLMs

5 – 7 weeks

The modern backbone of almost every NLP system.

BERT & Encoder Models

Masked language modelling, NSP, fine-tuning for classification & NER

🎬 HuggingFace NLP Course Ch. 1–4 (free)

GPT & Decoder Models

Causal LM, autoregressive generation, temperature, sampling strategies

🎬 Andrej Karpathy — Let's build GPT from scratch

Fine-Tuning & PEFT

Full fine-tuning, LoRA, QLoRA, instruction tuning with Alpaca / Dolly datasets

📚 HuggingFace PEFT library docs
04

Advanced LLM Applications

Ongoing

Production patterns for LLM-powered systems.

RAG Pipelines

Embedding models, vector stores (FAISS, Chroma), re-ranking, evaluation

RLHF & Alignment

Reward modelling, PPO on LLMs, DPO, constitutional AI

Evaluation & Benchmarking

BLEU, ROUGE, BERTScore, LLM-as-judge, LM-Eval-Harness

Agents that learn by doing — from tabular methods to deep RL systems that master games and control robots. Assumes ML Foundations is complete.

🕐 Est. 18–24 weeks
01

RL Fundamentals

4 – 6 weeks

The mathematical framework every RL algorithm is built on.

MDPs & Bellman Equations

States, actions, rewards, policies, value functions, discount factor γ

📚 Sutton & Barto — RL: An Introduction (free PDF)

Tabular Methods

Monte Carlo, TD-learning, Q-learning, SARSA, ε-greedy exploration

🎬 David Silver — UCL RL Lectures (YouTube)

OpenAI Gymnasium

Environment API, CartPole / MountainCar, observation & action spaces

📚 Gymnasium documentation
02

Deep Reinforcement Learning

6 – 8 weeks

Combine deep neural networks with RL to tackle high-dimensional environments.

Deep Q-Networks (DQN)

Experience replay, target network, Double DQN, Dueling DQN, PER

📄 Paper: "Human-level control through DRL" (Mnih et al. 2015)

Policy Gradient Methods

REINFORCE, actor-critic (A2C/A3C), advantage estimation, entropy bonus

🎬 Spinning Up in Deep RL — OpenAI

PPO & SAC

Clipped objective (PPO), entropy regularisation (SAC), continuous control

📄 Paper: "Proximal Policy Optimization" (Schulman et al. 2017) 📚 Stable-Baselines3 docs
03

Advanced RL Topics

5 – 7 weeks

Scalable and specialised techniques for real-world RL problems.

Model-Based RL

World models, Dyna architecture, MuZero, Dreamer

Multi-Agent RL

Cooperative vs competitive settings, MADDPG, QMIX, PettingZoo

Sim-to-Real Transfer

Domain randomisation, MuJoCo / Isaac Gym, real robot deployment

04

Applications & Research

Ongoing

Apply your skills to open problems in games, robotics, and optimisation.

Game Playing

Atari, Chess/Go with AlphaZero, custom competitive environments

Robotics

Locomotion, manipulation, navigation with physical or simulated robots

Taking models from notebook to production — versioning, pipelines, deployment, monitoring. Assumes ML Foundations is complete.

🕐 Est. 12–18 weeks
01

Engineering Foundations

2 – 3 weeks

Software engineering practices every ML engineer should know.

Git & GitHub for ML

Branching, pull requests, Git LFS for large files, CI basics

🎬 The Missing Semester of CS Education (MIT)

Python Packaging & Environments

venv, Poetry, pyproject.toml, structuring an ML project

Testing ML Code

pytest, data validation with Great Expectations, testing pipelines

02

Experiment Tracking & Data Versioning

3 – 4 weeks

Reproducible experiments and auditable datasets.

MLflow

Tracking runs, logging params/metrics/artifacts, model registry, UI

📚 MLflow documentation

DVC (Data Version Control)

Versioning large datasets & models with Git-like commands

📚 DVC Get Started guide

Weights & Biases (W&B)

Rich experiment dashboards, sweeps (hyperparameter search), artefacts

03

Model Serving & Deployment

4 – 6 weeks

Wrap your model in an API and ship it reliably to users.

FastAPI for ML APIs

Pydantic models, async endpoints, input validation, Swagger UI

📚 FastAPI official docs

Docker for ML

Multi-stage builds, GPU-enabled images (CUDA), slim production images

🎬 Docker for ML Engineers — blog series

Cloud Deployment

AWS SageMaker / GCP Vertex AI basics, serverless inference, autoscaling

04

Production ML Systems

Ongoing

Keep models healthy at scale: monitor, retrain, and iterate.

ML Monitoring

Data drift, concept drift, Evidently AI, Prometheus + Grafana dashboards

CI/CD for ML

GitHub Actions pipelines, automated retraining triggers, model promotion gates

Feature Stores

Feast, Tecton — sharing and serving features consistently across training & inference
